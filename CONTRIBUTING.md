Contributing
============

Thanks for your interest in improving SiteBlocker. A few quick guidelines:

- Bugs & issues: open an issue on GitHub describing the problem, browser/version, and steps to reproduce.
- PRs: fork the repo, create a descriptive branch, and open a PR against `main`.
- Tests: add Jest unit tests for any new pure logic in `src/` and run `npm test`.
- Formatting: keep changes small and focused; follow existing code style.

Local development

1. Install dev deps:

```bash
npm install
```

2. Run tests:

```bash
npm test
```

3. Load unpacked extension in Brave/Chrome via `brave://extensions` or `chrome://extensions`.

Contact

If you need to reach me about contributions, open an issue or add a comment on the PR.
